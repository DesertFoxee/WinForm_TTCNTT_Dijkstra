# Thực tập công nghệ thông tin

## Introduction :
Bài tập cuối kỳ học phần công nghệ thông tin . Bài toán tìm đường đi ngắn nhất sử dụng thuật toán Dijkstra
###### **Input** :Điểm bắt đầu , điểm kết thúc, số điểm , số đường (1 chiều hoặc 2 chiều hoặc không).
###### **Output** : Đường nối giữa hai điểm (được thể hiện trực quan). 
## Setup:
- Ngôn ngữ : C#
- GUI : Winform 
- IDE : Visual studio 2019
